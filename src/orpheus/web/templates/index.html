This tool derives private keys from wallet files. Run only on trusted, air-gapped machines. Clear scrollback and use the ephemeral Docker image when recovering wallets with value.

I.

Scan a directory

Upload one or more wallet files — wallet.dat, .wallet, .aes.json, JSON dumps, plain seed phrases. Orpheus dispatches each to the matching extractor and resolves balances via the chosen provider.
